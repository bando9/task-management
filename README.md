## Installation

Clone the repository and install dependencies:

```bash
# Clone the repo
git clone https://github.com/bando9/mindflow.git
cd mindflow

# Install dependencies
bun install
# or
npm install

# Run the development server
bun dev
# or
npm run dev
```
